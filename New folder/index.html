Kim Sanderson

Flutter developer | AI engineer

Description :

SwiftSend is a cutting-edge delivery application designed to streamline the delivery process for both businesses and consumers. In a fast-paced world where convenience is paramount, SwiftSend aims to provide a seamless experience for users to send and receive packages efficiently and reliably. Whether it's a small parcel or a large shipment, SwiftSend ensures that deliveries are made swiftly and securely.



Tech stack used :

  • Frontend:React,Native
  • Backend:Node.js,Express.js
  • Database:MongoDB
  • Cloud Services : AWS(Amazon Web Services) for scalability and readabiliy
  • Payment Gateway Integration: Stripe,PayPal

Features :

  • Real-time chat
  • Payment Gateway Integration
  • Authentication
  • Responsive Design
  • Scalable
